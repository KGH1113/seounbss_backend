Welcome to the SeounBSS Backend

This backend server allows you to make song requests, view the requested songs, request the suggestions and view the suggestions.

To make a song request, send a POST request to the /song-request endpoint with the following JSON payload:


{
 "name": "Your Name",
 "studentNumber": "Your Student Number",
 "songTitle": "Song Title",
 "singer": "Singer/Band Name"
}

To view the requested songs, send a GET request to the /view-request endpoint.

To view all of the requests from now on, send a GET request to /view-all-requests. The server will respond with a JSON array of all requested songs.




To make a suggestion request, send a POST request to the /suggestion-request endpoint the following JSON payload:


{
 "name": "Your Name",
 "studentNumber": "Your Student Number",
 "suggestion": "Suggestion content"
}

To view the suggestion, send a GET request to the /view-suggestion endpoint.
